How to Contribute

We welcome contributions from developers of all skill levels. Here's how you can get involved:

1

Find an Issue

Browse our GitHub issues to find something that matches your skills. We tag issues as "good first issue" for newcomers.

2

Fork the Repository

Create your own fork of the repository where you'll make your changes.

3

Make Your Changes

Implement your solution following our coding standards and guidelines.

4

Submit a Pull Request

Open a PR with a clear description of your changes. Our team will review it promptly.

All significant contributions will be rewarded with XBTCR tokens from our developer fund.
